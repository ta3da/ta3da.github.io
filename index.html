Basic HTML
Name Syntax Purpose
Opening <!DOCTYPE HTML> Used in the begining of an HTML document to let the browser know what language is being used
Making a comment <!- - Comment - -> This will not be coded. Instead, it can be used to write a note to whoever is reading the code
Title <Head> <Title>Hello</Title> </Head> Used to put a title in the tab on the browser.
Link <a href=“url”>Link text</a> Used to put a link on a page
Link to same page <a href=“#IDtag”>Link text</a> Used to link to a specific area on a the same page
Image, w/description <img src=“url” alt=“Image description” > Used to put an image on a site. The alt tag allows screen readers to identify the image
Image Link <a href=“url”> <mg src=“img url”> </a> Used to make an image a link
